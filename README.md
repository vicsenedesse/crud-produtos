# crud-produtos

### Este projeto é um CRUD de produtos contedo os atributos:

> Descrição;
> 
> Data da Compra;
> 
> Preço;
> 
> Categoria.


### As tecnologias utilizadas foram:

> Backend: Springboot
>
>Frontend: Angular
>
>Banco: MySql


### Instruções para configuração:

* Diretório Backend:

      sudo apt-get install mvn
      
      mvn clean install

* Diretório Frontend:

       npm install
       
       npm start
       
* Diretório Principal

      docker-compose up
